HPLC Data Using Old Column
==========================

This directory contains HPLC chromatograms data collected using the **old HPLC column.**.The data include calibration measurements,
ABSTR batch reaction studies,optimization experimentation and scale up under different operating conditions.


The folder structure is organized based on reaction duration, flow conditions,
concentration optimization, and scale-up studies.

Folder Structure and Description
--------------------------------

1 h Reaction
~~~~~~~~~~~~
This folder contains HPLC data from **ABSTR batch reaction performed for a definite number of reaction cycles completed within 1 hour.** for two different concentrations of the reactant.
(500mM and 815mM)

---

500mM
-----

this folder contains HPLC chromatograms for the 1 hour ABSTR operation when reactant concentration was 500mM with each bach operation time with 6 minute 40 second.


Each PDF file represents the reactant composition sampled after a specific reaction time.

File naming Conversion
~~~~~~~~~~~~~~~~~~~~~~

- 1.pdf -->sample collected after 400 sec of reaction

- 2.pdf -->sample collected after 800 sec of reaction

similarly upto 3600 second with the interval of 400sec

  concentrations.

Scale-Up
~~~~~~~~

This folder contains HPLC data from **scale-up experiments** performed for the
automated batch process using the old column.

- **Reactant Concentration** :500mM
- **Photon Flux used** :185 µmolsec⁻¹
- **space velocity**:15 min⁻¹ with total operation time of 1 hour where each batch cycle time was 6 minute 40 second

- The experiments were conducted at a larger operational scale.



File naming convention
~~~~~~~~~~~~~~~~~~~~~~

- 1.pdf -->sample collected after first batch cycle of the reaction

- 2.pdf -->sample collected after second batch cycle of the reaction 

similarly upto 9th batch cycle of the reaction

Std. Graph (Standard Calibration Data)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
This folder contains **standard calibration chromatograms** for the old HPLC
column.

- Each PDF corresponds to a peak area of a standard Benzyl Bromide solution with a known concentration.

- File names indicate the concentration value associated with the standard.

**Examples:**

- ``1.25.pdf`` → Standard solution with a concentration of **1.25 microgram per mL**  
- ``2.50.pdf`` → Standard solution with a concentration of **2.50 microgram per mL**  
- ``5.00.pdf`` → Standard solution with a concentration of **5.00 microgram per mL**


Notes
-----
- All data in this directory were collected using the old HPLC column.
- Experimental conditions other than those specified were kept constant unless
  stated otherwise.
